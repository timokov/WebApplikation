Neu! Sie können Ihre Kleiderspende nun registrieren!

Bitte wählen Sie dazu aus, ob Sie die gespendeten Altkleider an unserer Geschäftsstelle abgeben, oder diese von unserem Sammelfahrzeug bequem abholen lassen möchten.

Wichtige Informationen zum Vorgehen:
Bitte wählen Sie unten die gewünschte Option aus und klicken Sie auf "Weiter". Füllen Sie das erscheinende Formular vollständig aus und vergewissern Sie sich, dass Ihre Angaben korrekt sind. Drücken Sie anschließend auf "Spenden", um Ihre Spende zu registrieren. Danke, jede Spende hilft!

Ihre Vorteile auf einen Blick:

  • Flexible Wahl zwischen Abholung oder Übergabe.
  • Flexible Wahl des Krisengebietes, an die Sie die Kleider spenden möchten.
  • Pratische Angabe der Kleidungsstücke, die Sie spenden möchten - so wird ein reibungsloser Ablauf deutlich vereinfacht.
